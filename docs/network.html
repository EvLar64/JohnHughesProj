  • Orange: Ferris Bueller characters

This network displays the various uses of parenthetical dialogue descriptors (such as “excited”, “matter-of-fact”, etc.) across all three of the screenplays. Not only can you see which descriptors are used by which characters, but how they overlap between the movies (or remain completely isolated in the cases of more circumstantial descriptors, like “looks at the geek, incredulous”). Through seeing which kinds of descriptors are used more commonly than others (and which ones are used frequently across different types of movies), we can gain a bit of understanding about how successful writers effectively handle brief actions and instances of emotions during dialogue - all while maintaining script momentum.
